# 🚀 AI Resume & LinkedIn Optimizer (ATS Dashboard)

An AI-powered SaaS web application that helps users **optimize resumes**, **improve LinkedIn profiles**, and **increase ATS (Applicant Tracking System) scores** using modern AI models.

---

## 🔥 Features

### ✅ ATS Resume Dashboard

* 📊 ATS Score (0–100)
* 🧠 AI-generated resume summary
* 🛠 Improvement suggestions
* 🔑 Missing keywords detection
* 📄 Fully improved resume output

---

### ✅ LinkedIn Optimizer

* 💼 Powerful headline generation
* 🧾 Optimized “About” section
* 🧠 Suggested skills
* 📈 Profile improvement tips

---

### ✅ UI/UX Features

* 📌 Sidebar Navigation
* 📋 Copy to Clipboard button
* ⏳ Loading indicators (AI processing)
* 🎯 Clean and responsive UI

---

## 🛠 Tech Stack

### Frontend

* React.js
* Axios
* Tailwind CSS 

### Backend

* Node.js
* Express.js
* Google Generative AI (Gemini API)

## ⚙️ Installation

### 1️⃣ Clone Repository

```bash
git clone https://github.com/your-username/ai-resume-linkedin-optimizer.git
cd ai-resume-linkedin-optimizer
```

### 2️⃣ Backend Setup

```bash
cd server
npm install
```

Create `.env` file:

```env
PORT=5000
GOOGLE_API_KEY=your_api_key_here
```

Run backend:

```bash
npm run dev
```

---

### 3️⃣ Frontend Setup

```bash
cd client
npm install
npm start
```
